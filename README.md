📦 Smart Inventory Manager (MERN Stack)


📌 Overview
Smart Inventory Manager is a full-stack MERN application designed to manage products efficiently with complete CRUD operations.
Built using MongoDB, Express, React, Node.js (MERN Stack)
Provides real-time product management
Designed with clean UI and scalable backend APIs


🚀 Features
➕ Add new products
📋 View all products
✏️ Update product details
❌ Delete products


🔄 Real-time data updates using REST APIs
🛠 Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Tools: VS Code, Postman


📂 Project Structure
├── Backend/              # Node.js + Express APIs
├── Frontend/             # React application
├── README.md



⚙️ How to Run Locally
1. Install dependencies
npm install
2. Setup MongoDB
Create Database: inventoryDB
Collection: products
3. Run Backend
cd Backend
npm run server
4. Run Frontend
cd Frontend
npm start
